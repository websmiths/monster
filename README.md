# monster-truth

A single-page site about the health effects of Monster energy drinks on teenagers.

Built with plain HTML, CSS, and a touch of vanilla JS. No dependencies, no build step, no frameworks. Google Fonts loaded via CDN.

## Hosting on GitHub Pages

1. Create a new GitHub repo (can be private or public)
2. Upload the contents of this folder
3. Go to **Settings → Pages**
4. Set source to **Deploy from a branch → main → / (root)**
5. GitHub will give you a URL like `https://yourusername.github.io/monster-truth/`

That's it. The site will be live within a minute or two.

## Files

- `index.html` — the entire site (self-contained, no external assets)
- `README.md` — this file

## Fonts

Loads [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) and [DM Sans](https://fonts.google.com/specimen/DM+Sans) from Google Fonts CDN. Requires an internet connection to render correctly.
